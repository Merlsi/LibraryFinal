# LibraryFinalDjango Library Management System
This is a Django library project that allows users to manage books, authors, publishers, and borrowing operations. The system provides different roles, such as staff and user, each with specific permissions and functionalities. Staff members can perform CRUD operations on books, authors, and publishers, while users can only view the available books.

# Features
User Authentication: Staff and users can log in to the system using their credentials.
Staff Functionality:
Add, edit, and delete books: Staff members can manage the library's book collection.
Add, edit, and delete authors: Staff members can manage the authors' information.
Add, edit, and delete publishers: Staff members can manage the publishers' information.
Borrow books: Staff members can borrow books on behalf of students.
Fine calculation: If a student returns a book after the end time, a fine is calculated and recorded.
User Functionality:
View books: Users can browse through the available books in the library.

# Usage
Log in to the system as a staff member or user using the provided credentials.
Staff members can perform the following operations:
Add, edit, and delete books, authors, and publishers using the respective management pages.
Borrow books to students using the borrowing functionality.
Calculate fines for late book returns.
Users can view the available books by browsing through the library's collection.
